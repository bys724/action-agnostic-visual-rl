# Crop Ratio Study (2026-03-06)

EgoDex 원본 영상(1920x1080)에서 정사각형 크롭 시 좌우 비율 결정.

## 배경

- 원본: 1920x1080 → 정사각형(1080x1080)으로 크롭 후 256x256 리사이즈
- 좌우로 잘라내야 할 총 마진: 840px
- 센터 크롭(50:50)은 좌 420px, 우 420px 제거

## 비교한 비율

| 비율 | 좌측 제거 | 우측 제거 | 설명 |
|------|----------|----------|------|
| 50:50 (센터) | 420px | 420px | 기본 센터 크롭 |
| 55:45 | 462px | 378px | 약간 오른쪽 |
| 60:40 | 504px | 336px | 오른쪽 이동 |
| **65:35** | **546px** | **294px** | **최종 선택** |

## 결론

- **65:35 비율 채택**: 좌측을 더 잘라내고 우측을 더 살림
- EgoDex ego-centric 영상에서 오른손 작업 영역이 우측에 더 많이 분포
- 센터 크롭 대비 작업 영역을 더 잘 포착
